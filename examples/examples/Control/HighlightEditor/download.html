Distribution Package
Debian Debian packages for Sarge/Sid are made by Jos√© L. Redrejo.

To install them a new line is necessary to be added to the /etc/apt/sources.list file and you can use apt-get later. 

deb http://www.linex.org/sources/linex/debian/ cl gambas
Or just download the packages from http://www.linex.org/sources/linex/debian/dists/cl/gambas/binary-i386/
Fedora Gambas is maintained in 'Fedora Extras'. You can install it easily by using the following command: 
yum install gambas
Gentoo Gambas is in the portage tree. The ebuilds are maintained by Marius Mauch. So a simple emerge gambas is sufficient to install it.
Linex You will find packages of Gambas for Linex at www.linex.org
Mandriva It seems that the binary packages of the stable version made by Mandriva are now correct.

Go to www.kudla.org to download binary packages of the stable and development versions. They are made by Rob Kudla.

QiLinux You will find the binary packages at 
ftp://ftp.qilinux.it/pub/QiLinux/devel-contrib/RPMS/i586
Slackware You will find packages of Gambas for Slackware 10 at www.linuxpackages.net
SuSE The binary packages made by SuSE do not follow the specifications.
DO NOT USE THEM!

Instead, use the packages made by Guillermo Ballester Valor.

You can download source and binary packages for all recent versions of SuSE at: 

ftp://ftp.gwdg.de/pub/linux/misc/suser-gbv/rpms/
