Publication de Gambas 2.16

Les principaux changement et corrections de bugs sont les suivants :

  • Gambas se compile correctemnt avec libtool 2.x sur les distributions Ubuntu récentes.
  • CTRL + Page Précédente et CTRL + Page Suivante permettent de naviguer entre les différents onglets de l'EDI.
  • Les raccourcis clavier de l'éditeur de menu sont correctement sauvegardés.
  • L'instruction READ fonctionne maintenant correctement sur les systèmes 64 bits.
  • Le composant gb.pdf se compile correctement avec les versions récentes de poppler.
Veuillez vous reporter au ChangeLog complet pour les détails.
